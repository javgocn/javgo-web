# JAVGO Web

本项目用于分享日常工作中的相关技术文档，一起进步吧！

[**快速阅读 →**](https://www.javgo.cn)

[![](./public/site/site-show.png)](https://www.javgo.cn)

## 本地部署

首先运行如下命令安装相关依赖：

```bash
pnpm i
```

然后运行如下命令启动开发服务器，并访问 localhost:3000：

```bash
pnpm dev
```
